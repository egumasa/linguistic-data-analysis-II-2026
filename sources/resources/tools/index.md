### From Linguistic Data Analysis I

This course contrasts rule-based / feature-based analysis (lexical diversity, syntactic complexity) with the classifications an LLM *judges in context*. If you took **Linguistic Data Analysis I** — or want to try those feature-extraction tools (TAALED, TAALES, TAASSC, Japanese NLP) — see the [LDA I course website](https://egumasa.github.io/linguistic-data-analysis-I/).
