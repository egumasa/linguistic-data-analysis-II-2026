Optional, self-contained notebooks that are **not** part of the core in-class sequence — reference
material you can dip into for corpus building and lexical analysis. Each is Colab-ready.

- [Downloading Aozora Bunko](./dl_aozorabunko.ipynb) — pull a Japanese literary corpus.
- [Extracting Collocations](./extracting-collocations.ipynb) — build collocation lists from a corpus.
- [Parallel Analysis for lexical diversity](./ld_parallel_analysis.ipynb) — choose the number of
  components for lexical-diversity measurement.

For the guided course notebooks, see [Tutorials](../../notebooks/index.md); for the annotated datasets,
see [Datasets](../datasets/index.md).
