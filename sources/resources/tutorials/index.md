These are the **guided, in-class tutorials** — you run them together during the hands-on sessions,
opening each notebook directly in Colab (Tohoku Google account, no setup). They are reached from the
day's session page under *Slides & Colab*; this page is the topic-ordered index.

The pipeline is identical across the week — **load gold → format prompt → call model → evaluate →
inspect errors**. Only the *task* changes. The datasets behind these tutorials are catalogued under
[Datasets](../datasets/index.md).

- [Day 2 — Annotation, gold standards & evaluation](./tutorial-day2-annotation-eval.md) ·
  notebook [`02_gold_and_eval.ipynb`](./02_gold_and_eval.ipynb)
  — learn the pipeline on an easy-to-judge task (CEFR-SP).
- [Day 3 — Replicating Kim & Lu with open data](./tutorial-day3-move-replication.md)
  — the same pipeline on a hard-to-judge discourse task (RAAMove → CaRS-50).

When you are ready to run your own study, move on to the
[mini-project starter tracks](../datasets/mini-project-tracks.md).
