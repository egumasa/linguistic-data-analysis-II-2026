## Welcome

**Linguistic Data Analysis II** is a five-day intensive graduate course (Tohoku University, Graduate School of International Cultural Studies, Summer 2026) on using and critically evaluating Large Language Models for linguistic analysis — annotation scheme design, gold-standard dataset construction, prompt design, and evaluation (precision/recall/F1, confusion matrices). It is the sequel to [Linguistic Data Analysis I](https://egumasa.github.io/linguistic-data-analysis-I/).
